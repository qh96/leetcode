For example:


    A -&gt; 1
    B -&gt; 2
    C -&gt; 3
    ...
    Z -&gt; 26
    AA -&gt; 27
    AB -&gt; 28 
    ...


Example 1:


Input: &quot;A&quot;
Output: 1


Example 2:


Input: &quot;AB&quot;
Output: 28


Example 3:


Input: &quot;ZY&quot;
Output: 701
